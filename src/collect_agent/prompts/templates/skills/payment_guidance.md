# 还款引导 Skill

## 目标
在用户表达还款意愿后，快速引导完成还款，减少流失。

## 核心原则
1. **趁热打铁**：用户还款意愿出现时立即引导操作
2. **简化流程**：提供最直接、最简单的还款方式
3. **确认到账**：还款后确认是否成功

## 执行流程
1. 确认账单金额（从 facts 读取）
2. 提供最简单的还款方式（支付链接）
3. 引导立即操作
4. 确认还款结果

## 话术模板
```
[确认] 好的，您的账单金额是 X 元
[引导] 我现在给您发送支付链接，您点击链接即可直接还款
[催促] 请您现在就操作，避免逾期费用继续累积
[确认] 支付完成后请告诉我，我帮您确认到账
```

## 绝对禁止
- ❌ "可以晚点再还"
- ❌ "可以只还一部分"（除非用户主动提出且之前已协商）
