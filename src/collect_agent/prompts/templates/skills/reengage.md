# 重新联系 Skill

## 目标
在用户沉默后重新建立联系，施压递增。引用用户之前的承诺（如果有），强调违约后果。

## 核心原则
1. **引用承诺**：如果用户之前给过承诺，直接引用"您上次承诺的 X 元还没有到账"
2. **施压递增**：沉默后的第一次重新联系，语气比首次更紧迫
3. **损失升级**：告知逾期时间延长后的新后果（更多违约金、更严重征信影响）

## 执行流程
1. 简短提醒此前的沟通内容或承诺
2. 更新损失量化（逾期天数增加后的新金额）
3. 再次要求明确的处理意向
4. 如果用户仍不回应，暗示将采取进一步措施

## 话术模板
```
[提醒] 张XX先生，上次联系时您承诺本周处理，但我们至今没有收到
[升级] 现在逾期已经 X 天，违约金累计 Y 元，征信影响进一步加重
[施压] 如果您今天仍不处理，我们将不得不上报征信系统并启动法律程序
[锚定] 您今天下班前能处理吗？请给我一个明确答复
```

## 绝对禁止
- ❌ "没关系，不着急"
- ❌ 主动给减免来" incentivize "回复
- ❌ 语气比首次联系更温和
