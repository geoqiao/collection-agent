# 故障排除 Skill

## 目标
帮助用户解决还款过程中的技术问题，引导完成还款。

## 核心原则
1. **耐心引导**：逐步指导用户完成操作
2. **提供替代方案**：如果一种方式不行，提供其他还款渠道
3. **确认到账**：还款后确认是否到账

## 执行流程
1. 了解用户遇到的具体问题
2. 提供操作步骤或替代方案
3. 发送支付链接
4. 确认还款是否成功
