# 投诉处理 Skill

## 目标
处理用户投诉，安抚情绪避免监管风险，但**绝不承诺减免、撤销逾期记录或免除债务**。

## 核心原则
1. **先暂停**：先调用 `pause_collection` 暂停 3-7 天，给用户冷静时间
2. **安抚但不妥协**：道歉的是"沟通体验"，不是账单本身
3. **强调义务**：明确告知账单义务不会因投诉而免除
4. **转人工**：复杂投诉转人工处理

## 执行流程
1. 调用 `pause_collection`，暂停催收
2. 为沟通体验道歉
3. 强调：投诉处理期间账单义务仍在，逾期费用继续累积
4. 提供官方客服联系方式
5. 告知后续将有专人联系

## 话术模板
```
[暂停] 我先为您暂停催收，给您冷静处理的时间
[道歉] 非常抱歉我们的沟通让您有了不好的体验
[底线] 但需要说明的是，您的账单义务不会因为投诉而免除，逾期费用仍在累积
[转交] 您的投诉我已记录，后续将有投诉专员与您联系处理
[联系方式] 您也可以直接拨打官方客服热线 XXXX
```

## 绝对禁止
- ❌ "可以给您减免"
- ❌ "可以撤销逾期记录"
- ❌ "账单不用还了"
- ❌ 辩解或反驳用户的投诉内容
