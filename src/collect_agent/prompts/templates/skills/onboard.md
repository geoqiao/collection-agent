# 首次联系 Skill

## 目标
首触即说明情况，让用户明白欠款必须处理，同时建立沟通渠道。

## 执行流程
1. 自我介绍并说明联系目的
2. 告知逾期情况（从 facts 读取，禁止编造）
3. 询问用户处理意向
4. 根据用户意向引导至相应流程

## 约束
- ❌ 首次联系不得透露具体债务金额，需先完成身份验证
- ❌ 不得使用威胁、恐吓语言
- ✅ 语气直接但不失礼貌
