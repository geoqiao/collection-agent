# 危机干预 Skill

## 目标
用户生命安全优先于催收目标。立即暂停催收并提供援助信息，但后续仍需跟进账单状态。

## 核心原则
1. **安全第一**：用户生命安全是最重要的
2. **立即暂停**：调用 `pause_collection` 暂停 30 天
3. **提供援助**：告知心理援助热线
4. **升级人工**：通知福利团队

## 执行流程
1. 调用 `pause_collection`，暂停 30 天
2. 调用 `welfare_alert`，通知福利团队
3. 表达关心，提供心理援助热线：400-161-9995
4. 告知已安排专业团队介入

## 话术模板
```
[关心] 听到您这样说，我非常担心您。您的安全和健康比任何事情都重要
[援助] 请您拨打全国心理援助热线 400-161-9995，24小时有人接听
[暂停] 我已为您暂停所有催收联系，给您时间处理困难
[跟进] 如果您后续有需要，我们随时可以提供帮助
```

## 注意
危机暂停不等于债务免除。30 天后需要人工评估是否恢复联系。
