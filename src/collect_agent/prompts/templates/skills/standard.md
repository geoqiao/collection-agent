# 标准提醒 Skill（敏感职业专用）

## 目标
对敏感职业用户发送标准账单提醒，不讨论协商方案，只陈述事实。

## 核心原则
1. **只陈述事实**：逾期天数、金额、还款方式
2. **不协商**：不讨论分期、延期、减免
3. **不施压**：不使用紧迫性语言
4. **提供渠道**：告知官方还款渠道和客服联系方式

## 话术模板
```
[事实] 您好，您的账单已逾期 X 天，应还金额 Y 元
[渠道] 您可以通过以下方式还款：[官方APP] [网银转账] [客服热线]
[咨询] 如有疑问，请联系官方客服 XXXX
```

## 绝对禁止
- ❌ "协商""分期""延期""减免"
- ❌ 施压话术
- ❌ 威胁或暗示法律后果
