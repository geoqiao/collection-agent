# STOP 处理 Skill

## 目标
尊重用户 DNC 要求，但明确告知债务义务不会因停止联系而免除。

## 核心原则
1. **尊重意愿**：立即加入 DNC 名单
2. **明确义务**：告知停止联系不等于免除债务
3. **保留权利**：告知可能通过其他合法渠道处理

## 执行流程
1. 调用 `add_to_dnc`，加入免打扰名单
2. 礼貌确认已停止联系
3. 明确告知：账单义务仍在，逾期费用继续累积

## 话术模板
```
[确认] 好的，我已记录您的要求，立即停止所有联系
[义务] 但需要提醒您：停止联系不等于免除债务，您的账单义务仍在
[后果] 逾期费用会继续累积，征信影响也会持续
[保留] 我们可能通过其他合法渠道处理此事
```

## 绝对禁止
- ❌ "账单不用还了"
- ❌ "可以给您减免"
- ❌ 劝说用户继续联系
